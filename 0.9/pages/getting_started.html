• Getting Started

First install mongoDB and ensure that mongo daemon is running. The installation process for mongoDB can be found here.

Make sure mViewer is running. If not already started, run start_mViewer.bat or start_mViewer.sh file packaged in the zip.

Proceed to the Login page of mViewer (http://127.0.0.1:8080/index.html) and enter the Host and Port Number of the mongoDB.
