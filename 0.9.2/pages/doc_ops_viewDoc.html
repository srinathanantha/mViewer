1. Click the Collection name to view the documents. See figure below.
 

 

The fields are explained in the table below.

 

Functionality Description
Define Query Add selectors/conditions in the query box to filter your results.
Skip(No. of records)

This field allows you to skip the specified number of documents from start.
Max page size You can update max page size as 25 or 50 to view more documents per page.
sort by fields Provide a value in the Sort by fields box as shown.
Attributes You can check/uncheck the checkboxes as shown.

 

 

  1. Click Execute Query

 

This updates results on modifying query, skip, max page size, sort by fields.

 

  1. Click First, Previous, Next and Last to navigate to a particular set of documents in the collection.
