1. In the Query Executor box, you can do the following:

 

Functionality Description
Define Query You can run a find query by specifying the contents in the collector.
Attributes

You can select/unselect values for v, ns, key._id, name, key
Skip(No. fo records) You can specify the number of records you wish to skip so that limited records are on display.
Max page size You can specify the number of records you wish to be displayed least being 10.
Sort by fields You can specify the field Id details here.

 

  1. Click Execute Query

 

The JSON view is now refreshed with new values. See Figure below.
