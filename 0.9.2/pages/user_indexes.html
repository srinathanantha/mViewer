Users > Edit User

  1. In the JSON view, click edit at the respective user.

 

The Please enter user credentials box appears. See figure below.

 

  1. Perform the following:
  • Password: You can reset your password.
  • Read only: Check this box, if you wish to deny write access permissions.
  1. Click Submit

 

The JSON view is refreshed along with the new changes.

Users > Delete User

  1. In the JSON view, click delete for the respective user.

 

The Are you sure? dialog prompt appears. See figure below.

  1. Click Yes

 

The user is dropped and JSON view is refreshed along with new changes.

 

Users > Drop Users

  1. Hover over system.users
  2. Click Drop Users as shown in the figure below.

 

  1. The system displays a delete user prompt. See figure below.

  1. Click Yes

 

All the users are now dropped. The JSON view is refreshed with changes.

Indexes > View an Index

  1. In the Query Executor box, you can do the following:

 

Functionality Description
Define Query You can run a find query by specifying the contents in the collector.
Attributes

You can select/unselect values for v, ns, key._id, name, key
Skip(No. fo records) You can specify the number of records you wish to skip so that limited records are on display.
Max page size You can specify the number of records you wish to be displayed least being 10.
Sort by fields You can specify the field Id details here.

 

  1. Click Execute Query

 

The JSON view is now refreshed with new values. See Figure below.
