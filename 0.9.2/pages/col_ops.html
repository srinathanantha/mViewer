Collection operations

Database > Add Collection

  1. Hover on the Database name.
  2. From the menu, click Add Collection. See figure below.

  1. Enter the new collection name and click Submit. See figure below.

 

  1. Enter the following details:

 

Functionality Description
Name Type the new name of the collection
Capped

Tick this checkbox to create a fixed size capped collection
Size (in bytes) Type the size
Max documents Type the maximum no. of documents to be loaded

 

 

  1. Click Submit

 

The collection is now created.

Database > Delete (Drop) Collection

  1. Hover on the Collection name.
  2. From the menu, click Drop Collection. See Figure below.

 

  1. Click Yes on the confirmation alert to delete the collection. See figure below.

 

 

The collection along with all its documents will then be dropped.

View Collection Statistics

  1. Hover on the Collection name.
  2. From the menu, click Statistics. See Figure below.

The page displays statistics for the respective collection as shown.
