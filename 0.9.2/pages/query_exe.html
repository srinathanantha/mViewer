Query Executor

mViewer provides the facility of executing the queries right from it using Query Executor,
thus eliminating the dependency on MongoDB terminal to execute the queries.

 

Query Executor is provided at DB, Collection and GridFS and Users and Indexes levels.
Clicking on selected DB or collection or GridFS or Users and Indexes displays Query Executor on the right side of the page.
Query Executor displayed for a particular DB will also performs operations on other DB's or collections aswell..

Data Base - Query Executor

  1. Select and click on a DB
  2. By Default query for DB statistics will be displayed with results

 

 

Collections - Query Executor

  1. Select and click on a Collection.
  2. By Default sub query for that collection along with all the documents of that collection will be displayed

 

GridFS - Query Executor

  1. Select and click on a Grid Bucket.
  2. By Default sub query for that Grid Bucket along with all the Files of that Bucket will be displayed
