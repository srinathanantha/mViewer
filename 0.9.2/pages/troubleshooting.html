Use <localhost.localdomain> as the host name.


It is recommended to first have a look at your host file and use the first value.

You will be redirected back to login page automatically.


Corrupted session indicates that the token used for session tracking has been lost. No data at the DB end would be compromised.

You can open up an issue at https://github.com/Imaginea/mViewer

 

It is recommended that you attach the server logs to the issue.
